# Readme

## Installation
* Clone the repo
* Install app dependencies 
` npm i `
* Add variables to `.env` or create file `.env.local` and add them there

```
DWH_USER=""
DWH_PASSWORD=""
AFF_PORTAL_PASSWORD="" 
```

* `npm run start`
